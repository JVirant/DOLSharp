
# /minorelic Command
Commands.GM.MinoRelic.Description:			Various Commands for Minotaur Relics
Commands.GM.MinoRelic.Usage.Create:			'/minorelic create <Name> <Model> <Group/Self/Realm> <SpellID> <Effect / 159 = red, 160 = white, 161 = gold>' creates a new Relic with the given arguments;
Commands.GM.MinoRelic.Usage.MoveHere:		'/minorelic movehere' moves the relic to player x, y, z and saves the coordinates as Spawn;
Commands.GM.MinoRelic.Usage.Name:			'/minorelic name <Name>' changes the name;
Commands.GM.MinoRelic.Usage.Spell:			'/minorelic spell <SpellID>' assign SpellID to the Relic;
Commands.GM.MinoRelic.Usage.Model:			'/minorelic model <Model>' changes the Model;
Commands.GM.MinoRelic.Usage.Effect:			'/minorelic effect <Effect / 159 = red, 160 = white, 161 = gold>' changes the effect the player gets when he picks up the relic;
Commands.GM.MinoRelic.Usage.Info:			'/minorelic info' browses relic Informations;
Commands.GM.MinoRelic.Usage.DeSpawn:		'/minorelic despawn' despawns the Relic;
Commands.GM.MinoRelic.Usage.Remove:			'/minorelic remove' removes the relic from the World and Database;
Commands.GM.MinoRelic.Usage.XP:				'/minorelic xp <amount>' give amount xp to the Relic;
Commands.GM.MinoRelic.Usage.ShowAll:		'/minorelic showall' shows all Relics;
Commands.GM.MinoRelic.Usage.Spawn:			'/minorelic spawn <ID>' spawns the Relic with the ID;
