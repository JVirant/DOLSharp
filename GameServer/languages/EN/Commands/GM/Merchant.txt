# /merchant Command
Commands.GM.Merchant.Description:			Various merchant creation commands!
Commands.GM.Merchant.Usage.Create:			'/merchant create' to create an empty merchant;
Commands.GM.Merchant.Usage.CreateType:		'/merchant create <type>' to create an empty merchant with class typed;
Commands.GM.Merchant.Usage.Info:				'/merchant info' to show info about merchant;
Commands.GM.Merchant.Usage.Save:				'/merchant save' to save this merchant as new object in the DB;
Commands.GM.Merchant.Usage.Remove:			'/merchant remove' to remove this merchant from the DB;
Commands.GM.Merchant.Usage.Sell:				'/merchant sell add <itemsListTemplateID>' to assign this merchant with an articles list template;
Commands.GM.Merchant.Usage.SellRemove:		'/merchant sellremove' to remove the articles list template from merchant;
Commands.GM.Merchant.Usage.Articles.Add:		'/merchant articles add <itemTemplateID> <pageNumber> [slot]' to add an item to the merchant articles list template;
Commands.GM.Merchant.Usage.Articles.Remove:	'/merchant articles remove <pageNumber> <slot>' to remove item from the specified slot in this merchant inventory articles list template;
Commands.GM.Merchant.Usage.Articles.Delete:	'/merchant articles delete' to delete the inventory articles list template of the merchant;
Commands.GM.Merchant.Usage.Type:				'/merchant type <classtype>';
Commands.GM.Merchant.CommandOverview:		Type /merchant for command overview
